## About Prefix, Root and Suffix

e.g. supercalifragilisticexpialidocious

- super-
- cali-: beauty 
	- calisthenics 健美操
	- calligraphy 
- fragil-: delicate
	- fragment
	- fraction 片段
	- frail 脆弱的
- expia-: compensate
	- expiate(vt.) 赎罪
	- expiation
- doc-: teach
	- doctor
	- document
	- docile 温顺的，驯服的
	- doctrine 学说，教义
	- docent 讲解员

原意：用精致的美丽补偿可教育性的缺失

## BabyDLP

根据 Pohlig-Hellman 算法，我们可以将算法分为以下几步：

- 质因数分解 p - 1
- 检查 g 的阶，更新 order
- 再次质因数分解 order，得到 m 个因子组成的数组 pr
- 对于 pr 的每个 $p_i^{k_i}$ ，离散对数算出 $(g^x)^{\frac{p-1}{p_i^r}}\equiv a^{\frac{p-1}{p_i^r}}(mod\space p)$ 的解
- 得到 m 个关于 x 的式子，利用中国剩余定理计算出最后 x 的值

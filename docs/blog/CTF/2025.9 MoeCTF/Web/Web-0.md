## WriteUp

打开附件，看到一大堆奇奇怪怪的符号，直接丢给 AI 得知这是一个是一种特殊的 JavaScript 混淆方式，叫做 **JSFuck**。它使用只有6种字符 `[](){!+}` 就能构造出任意的 JavaScript 语句。题目提示“控制台”，明显是和**浏览器控制台**（Console）有关。

我们直接将这段奇怪符号复制到控制台即可输出结果：
